  ESC                   Exit the emulator
  F1                    Test Switch
  F2                    Service Switch
  F3                    Start 1
  F5                    Save State
  F7                    Load State
  F8                    Enable all layers
  F9                    Turn layer #0 on/off
  F10                   Turn layer #1 on/off
  F11                   Turn layer #2 on/off
  F12                   Turn layer #3 on/off

 Look at the source for informations.
